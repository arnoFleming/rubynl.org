Workshop JS in Rails

A brand new workshop for medior/ senior Rails developers about JavaScript in Rails: “Write Clean JavaScript in your Rails application”.

The goal is to write clean, beautiful JavaScript in Rails applications and become more confident in writing JavaScript in Rails. During this hands-on workshop, you will learn how to make your JS code as pretty as your Rails code and you will get the opportunity to refactor or create JS code in your own projects.
